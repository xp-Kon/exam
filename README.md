# 📚 马克思复习题 · 练习工具

纯前端单页应用，支持多科目题库的练习、模拟考试和错题管理。零构建工具，零依赖。

## ✨ 功能特性

| 功能 | 说明 |
|---|---|
| **题库浏览** | 分页浏览全部题目，支持关键词搜索 + 题型筛选（单选/多选/填空/判断）+ 随机排序 |
| **顺序练习** | 按题库顺序逐题练习，进度自动保存 |
| **随机练习** | 打乱题目顺序随机练习 |
| **题型专项** | 单选题 / 多选题 / 填空题 / 判断题 分开练习 |
| **模拟考试** | 10/20/40 题可选，支持限时模式 |
| **错题本** | 自动记录错题及错误次数，可集中回顾 |
| **我的收藏** | 收藏重点题目，方便复习 |
| **统计看板** | 正确率、练习进度、高频错题 TOP5 |
| **多科目切换** | 马克思主义原理、操作系统、数据库、JSP/Web 程序设计 |
| **触屏导航** | 移动端左右滑动切题 |
| **快捷键** | `A` `B` `C` `D` 选答案 · `←` `→` 翻题 · `S` 收藏 · `Enter` 提交填空 |
| **暗色主题** | 一键切换 light/dark 主题 |

## 🚀 快速开始

```bash
python -m http.server 8080
# 浏览器打开 http://localhost:8080
```

无需安装依赖，任何支持 Python 的环境均可运行。

## 🧩 技术栈

- **纯原生 JavaScript** — 无框架、无构建工具
- **Tailwind CSS**（CDN） — 基础布局
- **Font Awesome**（CDN） — 图标
- **localStorage** — 数据持久化

## 🔧 修改指南

- **增删题目**: 修改 `js/data.js` 中对应科目的 `questions` 数组（或通过 `excel2js.py` 从 Excel 生成）
- **新增科目**: 在 `data.js` 的 `SUBJECTS` 中添加键，在 `index.html` 的 `<select>` 中添加 `<option>`
- **修改样式**: 编辑 `css/style.css` 中的 CSS 变量
- **新增路由**: 在 `index.html` 侧栏加 `<a>`，在 `handleRoute()` 的路由表加映射，写对应 `renderXxx()` 函数

## 📝 题目数据格式

```javascript
// 选择题
{
  id: 1,
  question: "题目文字",
  options: { A: "选项A", B: "选项B", C: "选项C", D: "选项D" },
  answer: ["A"],          // 单选
  // answer: ["A", "C"],  // 多选
  type: "single",         // 或 "multiple"
  source: "来源"          // 可选
}

// 填空题
{
  id: 101,
  question: "______ 是马克思主义的基石。",
  answer: ["实践"],        // 单个空或多个空
  type: "fill",
  source: ""
}
```
